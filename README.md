# DirectVRM

DirectVRM は、.NET Framework で利用可能な [VRM](https://vrm.dev/) ライブラリです。

## リリース

リリースは NuGet 上で行っています。

[NuGet Gallery | DirectVRM<br/>![Nuget](https://img.shields.io/nuget/vpre/DirectVRM.svg)](https://www.nuget.org/packages/DirectVRM/)

## 動作環境

+ Windows 10 (x64 only)
+ .NET Framework 4.7.1+
+ DirectX 11.0+
